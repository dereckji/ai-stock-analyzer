# AI 股票分析助手 - Mac 版

## 三种使用方式（按推荐顺序）

### 方式一：双击 .command 文件（最简单，2 分钟上手）

1. 首次使用：右键 `启动.command` → 打开方式 → 终端
   - 首次必须用这种方式打开（macOS 安全机制）
   - 会自动装依赖（约 2-3 分钟）
2. 之后双击即可启动
3. 浏览器自动打开 `http://localhost:8501`

### 方式二：打包成 .app（最便利，桌面图标启动）

1. 终端运行 `./打包.command`
2. 等 3-5 分钟，会在 `dist/` 生成 `AI股票分析.app`
3. 把它拖到：
   - 桌面
   - 启动台 (Launchpad)
   - 应用程序文件夹

### 方式三：安装到应用程序（一键安装到 /Applications）

1. 先运行方式二打包出 .app
2. 再运行 `./安装到应用程序.command`
3. 在启动台搜「AI 股票」就能找到

## 重要提示

### macOS 安全机制

首次打开下载的 .command 或 .app 文件时，会提示「无法打开」。

**解决方法（任选一种）：**

**A. 系统设置法**（推荐）
1. 打开「系统设置」→「隐私与安全性」
2. 向下滚动到最底部
3. 点击「仍要打开」按钮

**B. 右键打开法**
1. 在 Finder 中找到文件
2. 右键点击 → 选择「打开」
3. 弹出确认框 → 点击「打开」

### 端口冲突

如果 8501 端口被占用，编辑 `app.py` 最后一行：
```python
streamlit run app.py --server.port 8502
```

### 防火墙提示

首次启动时 macOS 可能询问是否允许网络连接，点击「允许」（需要联网获取数据）。
