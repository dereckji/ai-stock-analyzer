# AI 股票分析助手 - 桌面版使用指南

## 三种使用方式

### 方式一：一键启动脚本（最简单，30 秒上手）

**适用人群**：愿意装一次 Python 的用户
**优点**：体积小、启动快、好维护
**缺点**：需要先装 Python

#### Windows 用户

1. 安装 Python（如果还没装）
   - 访问 https://www.python.org/downloads/
   - 下载 Python 3.11 或更高版本
   - **重要**：安装时勾选 "Add Python to PATH"
   - 装完重启电脑

2. 把整个 `ai_stock_analyzer` 文件夹放到桌面

3. 双击 `启动.bat`
   - 首次运行会自动安装依赖（约 1-2 分钟）
   - 之后会自动打开浏览器访问 http://localhost:8501

4. 浏览器中：
   - 左侧填入 DeepSeek API Key
   - 输入股票代码（如 000628）
   - 点击「🚀 开始分析」

#### Mac / Linux 用户

1. 确保装了 Python 3.8+
   ```bash
   python3 --version
   ```

2. 终端进入文件夹
   ```bash
   cd ~/Desktop/ai_stock_analyzer
   chmod +x 启动.sh
   ./启动.sh
   ```

---

### 方式二：打包成 EXE（最便利，双击就用）

**适用人群**：想完全摆脱 Python、给朋友用
**优点**：双击即用、不需要装 Python
**缺点**：单个文件夹约 200-300MB

#### Windows 打包

1. 在 `ai_stock_analyzer` 文件夹内双击 `打包成EXE.bat`
   - 自动安装 PyInstaller
   - 自动打包（约 3-5 分钟）
   - 完成后生成 `dist\AI股票分析\` 文件夹

2. 进入 `dist\AI股票分析\`
3. 双击 `AI股票分析.exe`
4. 浏览器自动打开 → 立即使用

#### Mac 打包

```bash
cd ~/Desktop/ai_stock_analyzer
./打包成APP.sh
```

完成后 `dist/AI股票分析/AI股票分析` 双击运行

---

### 方式三：创建桌面快捷方式（Windows）

如果你用方式一/二，想在桌面创建图标：

**方式一用户**（一键启动脚本）：
1. 右键 `启动.bat` → 发送到 → 桌面快捷方式
2. 在桌面右键这个快捷方式 → 属性
3. 可以改图标（需要 .ico 文件）

**方式二用户**（EXE）：
1. 进入 `dist\AI股票分析\`
2. 右键 `AI股票分析.exe` → 发送到 → 桌面快捷方式

## 常见问题

### Q1: 双击启动.bat 闪退？
**A**: 打开 CMD 命令行，手动运行看错误信息：
```cmd
cd 路径\ai_stock_analyzer
启动.bat
```

### Q2: 端口 8501 被占用？
**A**: 编辑 app.py 最后一行，改为：
```python
streamlit run app.py --server.port 8502
```

### Q3: akshare 安装失败？
**A**: 可以不改，正常使用。代码会自动用演示数据。但要真实行情就必须装。

### Q4: 打包后杀毒软件报警？
**A**: PyInstaller 打包的 EXE 经常被误报。点击「仍要运行」或加入白名单。

### Q5: 启动慢？
**A**: 首次启动要加载数据，正常。后续会缓存，5-10 秒即可。
